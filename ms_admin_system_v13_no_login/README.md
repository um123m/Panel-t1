# Ms Admin System V11

تعديل جديد حسب الصور الأخيرة:

- صفحة **الرئيسية** أعيد ترتيبها حتى تكون مثل صورة الفيديو: الترحيب يمين، الإحصائيات بالنص، الاقتراحات يسار، وتحتها الجداول بنفس الترتيب.
- صفحة **إدارة اللاعبين** صارت أقرب للصورتين: كرت البحث، التنبيه الأحمر، حقيبة اللاعب، معلومات اللاعب، أزرار الإدارة، صورة اللاعب، مركبات اللاعب، والملاحظات.
- تم وضع صورة اللاعب من اللقطة التي أرسلتها داخل كرت **صورة اللاعب**.
- السايدبار بدون نظام الأسهم، وكل العناصر ظاهرة مباشرة.
- تسجيل الدخول يبقى محفوظ لمدة سنة: إذا دخلت Email/Password مرة، ما يحتاج تعيدها كل مرة.
- بيانات الدخول:

```text
Email: Ms
Password: Ms
```

## التشغيل

```bash
npm start
```

أو:

```bash
node server.js
```

بعدها افتح الرابط الذي يظهر لك في CodeSandbox / Replit / Render / أي استضافة Node.js.

## الملفات المهمة

- `server.js` السيرفر والـ API الحقيقي.
- `data/db.json` قاعدة البيانات المحفوظة.
- `public/app.js` منطق الواجهة.
- `public/styles.css` شكل الواجهة.
- `public/ms_logo.png` اللوغو.
- `public/player_scene.jpg` صورة اللاعب داخل صفحة الإدارة.
